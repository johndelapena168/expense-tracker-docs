# Profile Page

This document describes the Profile page.

## Purpose

Display user information and account options.

## Sections

### Profile Header
- Avatar placeholder
- User name
- Account type

### Quick Actions
- Categories shortcut
- Settings shortcut
- Notifications toggle
- Dark mode toggle

### Support Links
- Help & Support
- Privacy Policy
- Terms of Service

### Account Actions
- Logout button

## UI Layout

- Fixed header with back
- Profile card
- List of options
- Toggle switches
- Bottom navigation

## State Management

Uses AppContext for:
- Dark mode state
- Theme toggling

## Future Enhancements

- Avatar upload
- Profile editing
- Account settings
- Password change
