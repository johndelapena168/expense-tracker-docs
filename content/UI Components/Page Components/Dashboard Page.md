# Dashboard Page

This document describes the Dashboard page.

## Purpose

Primary analytics hub showing financial overview.

## Sections

### Total Spending Card
- Current month total
- Gradient background
- Large formatted amount

### Statistics Grid
- Transaction count
- Active categories
- Budget utilization

### Category Breakdown
- Progress bars per category
- Spent vs budget
- Color-coded

### Payment Methods
- Cash amount
- Credit Card amount
- GCash amount

### Monthly Trend
- 12-month bar chart
- Current month highlight
- Visual comparison

### Recent Transactions
- Last 5 expenses
- Category icons
- Quick amounts

## Data Processing

Computed from contexts:
- Current month expenses
- Category aggregations
- Payment method totals
- Monthly trends

## UI Layout

- Scrollable content
- Card-based sections
- Responsive grid
- Gradient accents
