# Page Components

This document provides an overview of the page components.

## Page Structure

All pages follow a consistent layout:
- Fixed header with navigation
- Scrollable content area
- Bottom navigation bar

## Pages

### Dashboard
Financial overview with:
- Total spending card
- Category breakdown
- Payment method distribution
- Monthly trend chart
- Recent transactions

### Expenses
Expense list with:
- All expenses chronologically
- Category icons
- Formatted amounts
- Delete actions

### Add Expense
Form for creating expenses:
- Amount input
- Date picker
- Description field
- Category selection
- Payment method

### Categories
Category management:
- List of categories
- Add/edit/delete
- Icon and color selection
- Budget setting

### Reports
Analytics visualization:
- Category spending summary
- Monthly trend chart
- Statistics cards

### Settings
App configuration:
- Currency selection
- Dark mode toggle
- Notifications
- About section

### Profile
User account:
- Avatar placeholder
- Quick actions
- Navigation links

See individual page documentation for details.
