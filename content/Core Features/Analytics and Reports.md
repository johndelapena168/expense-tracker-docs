# Analytics and Reports

This document describes the analytics and reporting features of the Expense Tracker.

## Dashboard Analytics

### Financial Overview Cards
- Total spending for current month
- Transaction count
- Active categories count
- Budget utilization percentage

### Category Breakdown
- Spending per category
- Budget vs. actual comparison
- Progress bars for visual representation
- Color-coded categories

### Payment Method Analysis
- Distribution across Cash, Credit Card, and GCash
- Amount spent per method
- Percentage breakdown

### Monthly Trend Chart
- 12-month spending history
- Current month highlighting
- Bar chart visualization
- Year selection

## Reports Page

### Category Spending Summary
- Sorted list by spending amount
- Percentage of total spending
- Visual progress indicators

### Monthly Comparison
- Month-over-month trends
- Year-over-year analysis
- Average spending calculations

## Data Processing

All analytics are computed client-side:
- Filter expenses by date range
- Aggregate by category
- Calculate percentages and totals
- Format currency based on user preference
