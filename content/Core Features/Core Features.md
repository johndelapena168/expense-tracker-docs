# Core Features

This document provides an overview of the core features of the Expense Tracker application.

## Feature Overview

### Expense Management
- Add new expenses with amount, description, category, date, and payment method
- View all expenses in a scrollable list
- Delete expenses with swipe or button action
- Automatic categorization and icon assignment

### Category Management
- Create custom spending categories
- Assign icons and colors to categories
- Set monthly budgets per category
- Edit and delete existing categories

### Analytics and Reports
- Dashboard with financial overview
- Monthly spending trends
- Category breakdown with progress bars
- Payment method distribution

### Settings and Profile
- Currency selection (8 currencies supported)
- Dark mode toggle
- Notification preferences
- User profile management

## User Interface

### Mobile-First Design
- Bottom navigation for easy access
- Touch-friendly interface elements
- Responsive layouts for all screen sizes
- Smooth animations and transitions

### Navigation Structure
- Dashboard (Home)
- Expenses List
- Add Expense (Floating Action Button)
- Reports
- Profile/Settings
